{% endmacro %} {% call faq_item(1, 'Apa itu KamulyoPelayanan?') %} KamulyoPelayanan adalah sistem pelayanan masyarakat berbasis tiket yang dirancang untuk memfasilitasi masyarakat dalam membuat pengaduan atau pengajuan dalam Desa Jatimulyo, Kabupaten Bantul, Daerah Istimewa Yogyakarta. Sistem ini memungkinkan warga untuk mengajukan masalah atau kebutuhan mereka melalui tiket berisi informasi seperti nama, nomor WhatsApp (yang akan diverifikasi melalui OTP), subjek, sarasi, dan sampiran jika diperlukan. {% endcall %} {% call faq_item(2, 'Bagaimana cara mengajukan tiket pengaduan?') %} Untuk mengajukan tiket pengaduan di KamulyoPelayanan, ikuti langkah-langkah berikut:
  1. Buka platform KamulyoPelayanan.
  2. Pilih opsi "Buat Tiket" pada halaman utama.
  3. Isi informasi yang diminta. Pastikan Anda mengisi informasi dengan benar dan lengkap.
  4. Setelah semua informasi terisi, kirim tiket Anda dengan menekan tombol "Kirim".
  5. Anda akan menerima konfirmasi melalui WhatsApp terkait status tiket Anda dan langkah-langkah selanjutnya.
{% endcall %} {% call faq_item(3, 'Apakah setiap pengajuan memerlukan lampiran?') %} Tidak, lampiran hanya diperlukan jika dibutuhkan untuk mendukung pengaduan atau pengajuan Anda. Jika masalah Anda dapat dijelaskan dengan baik tanpa lampiran, itu sudah cukup. Namun, jika Anda merasa lampiran seperti foto atau dokumen akan membantu menjelaskan situasi dengan lebih baik, Anda dianjurkan untuk menyertakannya. {% endcall %} {% call faq_item(4, 'Berapa lama saya harus menunggu respons setelah mengajukan tiket?') %} Waktu respons dapat bervariasi tergantung pada kompleksitas masalah dan antrian tiket. Tim KamulyoPelayanan akan berusaha sebaik mungkin untuk memberikan respons sesegera mungkin (kami akan mengusahakan agar setiap tiket bisa diselesaikan dalam jangka waktu 1x24 jam). {% endcall %} {% call faq_item(5, 'Bisakah saya mengajukan permintaan non-pengaduan melalui sistem ini?') %} Tentu saja! Selain pengaduan, KamulyoPelayanan juga menerima pengajuan permintaan seperti izin tertentu, permohonan pelayanan, atau hal lain yang memerlukan tindakan dari pihak desa. Pastikan Anda memilih "Pengajuan" di opsi Jenis ketika membuat tiket. {% endcall %} {% call faq_item(6, 'Apakah saya bisa melihat riwayat semua tiket yang pernah saya ajukan?') %} Tidak. KamulyoPelayanan didesain untuk bisa bekerja dengan cara yang sederhana. Tiket Anda akan diberikan nomor ID yang bisa Anda gunakan untuk melacak status tiket. Anda juga akan diberikan notifikasi melalui pesan WhatsApp ke nomor yang Anda gunakan untuk membuat tiket terkait perkembangan tiket. {% endcall %}
